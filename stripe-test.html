How It Works

  1. Click the button above
  2. Frontend calls POST /api/stripe/test
  3. Backend creates a Stripe Checkout Session
  4. Returns checkout_url in response
  5. Browser redirects to Stripe's hosted checkout page
  6. Enter test card details (see box above)
  7. Complete payment
  8. Redirects to stripe-success.html or stripe-cancel.html

Note: This is running in test mode. No real charges are made. Check payments at Stripe Dashboard.
